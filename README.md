# CV

My CV/resume, written in LaTeX, available in French and English.

- 🇫🇷 [CV (Français)](CS/CV_fr.pdf)
- 🇬🇧 [CV (English)](CS/CV_en.pdf)

## Build

The sources live in [CS/](CS/). To regenerate the PDFs:

```sh
cd CS
xelatex CV_fr.tex
xelatex CV_en.tex
```
